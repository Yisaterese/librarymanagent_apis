# Library Management API

This project is a Library Management API that allows users to manage books, authors, and library members. The API provides endpoints for creating, reading, updating, and deleting records.

## Features

- Add and manage books
- Add and manage authors
- Add and manage library members
- Search for books by title, author, or genre

## Technologies Used

- Java
- Spring Boot
- PostgreSQL
- Docker
- Cloudinary
